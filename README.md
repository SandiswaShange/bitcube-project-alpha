# Conference Room Booking System

## Developer onboarding (In Progress)

The purpose of this section is to help new contributors understand:

- What the system is intended to become
- How documentation is organised
- Where to find key project artefacts

The Conference Room Booking System is designed to help organisations manage meeting room reservations efficiently. The system supports employees, receptionists, and administrators by improving scheduling, reducing booking conflicts, and making room management easier.

## System Context

This system is a conference room booking system that handles:

- Employee room bookings
- Recurring booking requests
- Room availability checking
- Room searching based on equipment
- Room filtering by capacity
- Receptionist bookings for visitors
- Admini bookings view and report generation

The system currently uses:

- Frontend: C#/ASP.NET
- Backend: C#/ASP.NET
- Database: MySQL
